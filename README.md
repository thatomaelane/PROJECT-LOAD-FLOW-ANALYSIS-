# Power Flow Analysis of an IEEE 5-Bus Power System Using PowerWorld Simulator

## Overview

This project investigates the steady-state performance of an IEEE 5-Bus Power System using PowerWorld Simulator. Four case studies were developed to evaluate the impact of increased loading and different compensation techniques on voltage regulation, power flow, and overall system stability.

The project demonstrates how various compensation methods can improve voltage profiles and system performance under heavily loaded operating conditions.

---

## Case Studies

### Case Study 1 – Increased Load

- Doubled all system loads (100% increase)
- Analysed voltage profile degradation
- Identified the weakest bus in the network

### Case Study 2 – Shunt Compensation

- Installed a shunt capacitor at Bus 5
- Improved voltage regulation
- Reduced reactive power demand on the generators

### Case Study 3 – Series Compensation

- Added series compensation on Transmission Line 2–5
- Reduced line reactance
- Increased power transfer capability
- Improved overall voltage profile

### Case Study 4 – Coal Power Plant Integration

- Added a new coal power plant at Bus 5
- Supplied both active and reactive power
- Achieved full voltage recovery
- Compared technical and economic performance with capacitor-based solutions
